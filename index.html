lminimumcover

This is a simple web app built on top of Google App Engine that takes as input a set of functional dependencies, and returns a calculated L-minimum cover to this set.

Enter your set in the form field, in the syntax illustrated below:

{A->X,B->X,A->B,B->A,A->E,B->E,D->E,E->D,E->H,EF->G,I->Y,I->E,LMIO->K,BLMIO->P}

The code for this app is available from https://github.com/hammar/lminimumcover.
